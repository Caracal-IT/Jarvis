# Jarvis - Iron Man Inspired Assistant

This is an Android application with a Jarvis-themed interface, featuring Iron Man colors and design patterns.

## 🤖 AI Development Guidelines

This project has strict guidelines for AI-generated code and assets to maintain quality and thematic consistency.
Please refer to [`docs/standards/ai-assistant-instructions.md`](docs/standards/ai-assistant-instructions.md) for the full rules on:
1.  **Best Practices** (Kotlin/Android standards)
2.  **Resource Management** (Strict usage of `strings.xml`, `colors.xml`, etc.)
3.  **Design Patterns** (MVVM, Clean Architecture)
4.  **Maintainability** (Documentation, Modularity)
5.  **Asset Coherence** (Iron Man Red/Gold theme, Vector graphics)

> AI assistants are automatically directed to this file via `.github/copilot-instructions.md` (GitHub Copilot)
> and `.idea/ai-context.md` (JetBrains AI Assistant / Android Studio).

**All contributors and AI assistants must follow these instructions.**
